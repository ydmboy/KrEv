# KrEv

`KrEv` is the main Visual Studio solution for this workspace.

## Projects

- `KrEv/`: kernel-mode driver project
- `KrEvClient/`: MFC client that talks to the driver
- `shared/`: shared IOCTL and message definitions used by both sides

## Open

Open `KrEv.sln` in Visual Studio.

## Output

Build outputs are generated under `build/` and are ignored by git.
